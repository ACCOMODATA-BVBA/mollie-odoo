Official Mollie Module

Sync mollie settlements with accounting journals.

This module needs Odoo Enterprise to work.

How to configure?

  • Install the module.
  • Open Journal Settings
  • Open Bank Account tab
  • Select Bank feeds = Mollie Synchronization
  • Add your mollie api key.

How to use it?

  • After configuring journal, go to accounting dashboard.
  • You will see, Sync Mollie button. Click on it.
  • This will open the dialog with all latest settlement
  • Choose the settlement to sync and click Sync settlement button the settlement.
